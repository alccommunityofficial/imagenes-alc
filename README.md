# ALC Assets Manager 🗃️

Este repositorio fue creado de manera muy rápida (en apenas 2 a 4 minutos) con un único y simple propósito: **gestionar y alojar archivos e imágenes**.

Aprovechando la infraestructura gratuita que nos ofrece Git y GitHub, utilizamos este espacio como un repositorio central para guardar nuestras imágenes, logos y recursos. De esta manera, obtenemos enlaces directos y permanentes (mediante GitHub Pages u otros despliegues estáticos) para luego utilizarlos libremente en los diferentes proyectos y sistemas de nuestro servidor, sin depender de alojamientos de imágenes de terceros que puedan expirar.

### ¿Para qué sirve?
- 🖼️ Alojamiento estático y seguro de imágenes (`/public/images/`, etc).
- 🔗 Proveer enlaces constantes para usarlos dentro de otros scripts de MTA, Discord Bots o sitios web externos.
- ⚡ Actualización rápida utilizando simple control de versiones.
